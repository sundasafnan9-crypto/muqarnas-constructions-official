# 🏗️ Muqarnas Constructions - Client Project

This is a premium, professional business website developed for a construction and architectural firm. It was built following specific client instructions to create a bold, historical-yet-modern digital presence.

## 🌟 Project Overview
Muqarnas Constructions is a real-world project designed to showcase structural elegance and architectural services. The design philosophy follows the tagline: *"Inspired by history, designed for tomorrow."*

## ✨ Key Features
- **Modern Dark UI:** A high-impact dark theme with gold and white accents for a luxury feel.
- **Pure CSS Layout:** High-performance styling without heavy scripting.
- **Client-Centric Design:** Developed strictly according to client specifications for layout and branding.
- **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
- **Professional Typography:** Bold headings and clear visual hierarchy.

## 🛠️ Built With
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styling, Flexbox/Grid layouts, and smooth animations.

## 👩‍💻 My Role
- **Front-end Developer:** Translated client requirements into a functional web interface using clean HTML/CSS.
- **UI/UX Designer:** Focused on visual impact and brand identity.
